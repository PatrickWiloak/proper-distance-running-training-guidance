```

│

│  │ Volume: ▲▲▲ Increasing
│  │ Intensity: ▼▼ Low
│  └─ Goal: Build aerobic foundation
│
├─ Phase 2: BUILD (8-12 weeks)
│  │ Volume: ▲ Maintain/Slight increase
│  │ Intensity: ▲▲ Add tempo work
│  └─ Goal: Develop lactate threshold
│
├─ Phase 3: PEAK (6-8 weeks)
│  │ Volume: ═ Maintain
│  │ Intensity: ▲▲▲ High (tempo + intervals)
│  └─ Goal: Race-specific fitness
│
├─ Phase 4: TAPER (1-3 weeks)
│  │ Volume: ▼▼▼ Decrease 30-50%

│  └─ Goal: Fresh for race
│
└─ Phase 5: RECOVERY (2-4 weeks)

   │ Intensity: ▼▼▼ None
   └─ Goal: Physical & mental reset
```

## The 3-to-1 Week Progression

Most effective progression: 3 weeks build, 1 week recovery

```
Week 1: Build    ████████ (40 miles)
Week 2: Build    ██████████ (45 miles)
Week 3: Build    ███████████ (48 miles)
Week 4: RECOVERY ██████ (32 miles) ← Down week
Week 5: Build    ██████████ (45 miles)
Week 6: Build    ███████████ (50 miles)
Week 7: Build    ████████████ (53 miles)
Week 8: RECOVERY ███████ (38 miles) ← Down week
```

**Why this works:**
- Allows adaptation from previous 3 weeks
- Prevents cumulative fatigue
- Reduces injury risk
- Mental break from building

## Quick Decision Guide

### "What phase am I in?"

**No goal race yet?** → BASE phase
- Build aerobic foundation
- Increase mileage gradually
- 80%+ easy running

**12-20 weeks until race?** → BUILD phase
- Add tempo runs (1x/week)
- Maintain volume
- Build lactate threshold

**6-10 weeks until race?** → PEAK phase
- Add intervals (every 7-14 days)
- Race-specific work
- 2 quality sessions/week

**1-3 weeks until race?** → TAPER phase
- Cut volume 30-50%
- Maintain some intensity
- Prioritize rest

**Just raced?** → RECOVERY phase
- Easy running only
- 2-4 weeks off hard training
- Let body repair

## Common Periodization Mistakes

❌ **Skipping base phase** → Jumping to hard workouts without foundation
✅ **Build base first** → 12-16 weeks minimum

❌ **No recovery weeks** → Building indefinitely without down weeks
✅ **3 build, 1 recovery** → Every 4th week reduce volume 20-30%

❌ **Constant intensity** → Always doing hard workouts
✅ **Progressive phases** → Build intensity gradually over months

❌ **Inadequate taper** → Training hard until race week
✅ **Proper taper** → Reduce volume 30-50% for 1-3 weeks

❌ **No off-season** → Racing and training hard year-round
✅ **Planned recovery** → 2-4 weeks easy after goal races

## Key Principles

1. **Volume before intensity** → Build mileage base before adding hard workouts
2. **Progressive overload** → Gradually increase stress over time
3. **Recovery weeks essential** → Down weeks allow adaptation
4. **Race-specific training** → Peak phase matches race demands
5. **Taper = trust your training** → Less is more in final weeks
